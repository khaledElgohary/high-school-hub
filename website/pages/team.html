Our Team

Meet the Highschool Hub development team.

  Ahmed

One of the main things I learned while developing this project is the Software Development Life Cycle. That, in tandem with AGILE and applying SOLID principles, gave me a much better understanding of what goes into development. Furthermore, I learned Version Control and branching strategies which taught me how to work on a large project being developed by many people. I also learned how to work with generated code, XML files and layout managers when designing the UI for the app. Another thing this project taught me is layered architecture which vastly improved my ability to functionally decompose large domain problems into small, cohesive and low coupling artifacts. This project has improved my skills as a developer in several ways.

  Braden

Some of the skills that I developed while creating the app is being able to apply SOLID design principles while creating the different classes. This is a very important skill as SOLID design principles enable you to write clean code that is easy to maintain as well as extendable. I have also developed skills in using Test Driven Development (TDD). Before creating this application, I never used TDD before, but I found that while I used it in this project, I was writing code that is simpler, easier to understand, as well as easy to maintain.
